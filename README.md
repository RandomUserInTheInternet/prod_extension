## Features

- ✅ Browse popular anime
- ✅ Browse latest updates
- ✅ Search anime
- ✅ View anime details and episodes
- ✅ Stream videos (m3u8/mp4)
- ✅ Sort by: Recent, Views, Rating, A-Z
- ✅ Multi-server support
- ✅ Sub/Dub selection
